# Telliot Core

Telliot is a command line tool and software development kit (SDK) for interacting with the [Fetch protocol](https://docs.fetchoracle.com/fetch/whitepaper/introduction).

- [Documentation](https://fetchoracle.github.io/telliot-core/)
- [Contributing](https://fetchoracle.github.io/telliot-core/contributing/)

To become a Fetch reporter, begin by installing the
[telliot-feeds](https://github.com/fetchoracle/telliot-feeds) package.
